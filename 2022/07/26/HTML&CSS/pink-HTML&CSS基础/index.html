CSS

选择器

选择器包括：基础选择器和复合选择器两大类。

基础选择器

由单个选择器组成。包括：标签选择器，类选择器，id选择器和通配符选择器。

标签选择器

某一类标签全部选择出来。

类选择器

单独选一个或几个。

多类名

一个标签可以设置两个或多个类名。中间空格隔开。

id选择器

样式#定义，结构id调用，且只能调用一次。一般和js搭配。

通配符选择器

选择所有标签。

字体属性

style 属性 属性值
font-family 字体 多种字体，逗号隔开
font-size 字号 px;
font-weight 粗细 normal(400) bold粗 bolder特粗 lighter细 自定义不用单位(100~900)
font-style 样式 normal italic斜体

字体复合属性

文本属性

style 属性 属性值
color 字体颜色 :rgb(200,0,0)RGB代码 :red预定义 :#FF0000十六进制
text-align 水平对齐 left right center
text-decoration 文本装饰 none underline下划线 overline上划线 line-through删除线
text-indent 文本缩进 px; em;(相对元素，一个当前文字长度)
line-height 行间距 =上间距+文本高度+下间距 px;
  • text-decoration: none;可以用来取消超链接下划线。

引入方式

样式表

内部样式表

<style> </style>理论上可以放到HTML任意地方，但是一般会放在文档的<head>标签中。

行内样式表

< style="color: red;"></>只能控制当前元素，简单修改。

外部样式表

样式单独写在CSS文件中，之后把CSS引入。

  1. 新建 .css 文件，把css代码放入。
  2. 在HTML中用<link>标签引入css文件。放在头文件里<link rel="stylesheet" href="文件名.css">

Emmet语法

快速生成HTML结构语法（idea可用）

  1. 标签 + tab
  2. 标签*n + tab 快速生成n个标签
  3. 父子级，用> + tab
  4. 兄弟级，用+ + tab
  5. 生成带类名或id的，.demo或者#demo + tab
  6. .demo$*5 + tab 快速生成 class="demo1",,,"demo5"
  7. 标签{内容}*5 快速生成5个标签含内容

复合选择器

后代选择器

元素1(父) 元素2(子/任意级别后代) {样式声明}。

子选择器

最近一级子元素。元素1(父)>元素2(子/任意级别后代) {样式声明}。

并集选择器

用于选择多组标签，用“,”表示&。

伪类选择器

链接伪类选择器
说明
a:link 选择所有未被访问的链接
a:visited 选择所有已被访问的链接
a:hover 选择鼠标指针位于其上的链接
a:active 选择活动链接（鼠标按下未弹起链接）

四个选择器的顺序不能改变。（LoVa HAte）

focus伪类选择器

用于选取获得焦点的表单元素。焦点就是光标，一般情况<input>表单元素才能获取。

元素显示模式

块元素

独占一行。高宽行内行外边距可以控制。包括<div>,<p>,<h1>~<h6>,<ul>,<ol>,<li>等。

  • 文字内元素不能使用块级元素

    <p>内不能放块级元素，<h1>~<h6>文字块级也不能放块级。

行内元素

相邻行内元素在一行上，一行可以显示多个。高宽直接设置是无效的。默认宽度就是它本身内容的宽度。行内元素只能容纳文本或其他行内元素。包括<span>,<a>,<strong>/*文本加粗*/,<b>/*文本加粗*/,<em>/*斜体*/,<i>/*斜体*/,<del>/*删除线*/,<s>/*删除线*/,<ins>/*下划线*/,<u>/*下划线*/等。

  • 链接里不能再放链接
  • <a>里可以放块级元素，但是<a>转换一下块级模式最安全

行内块元素

<img />,<input />,<td>。一行可以放多个。默认宽度为本身内容的宽度。高宽行内行外边距可以控制。

显示模式的转换

说明
display: inline; 转换为行内元素
display: inline-block; 转换为行内块元素
display: block; 转换为块元素

背景

背景颜色

background-color: transparent(默认) | color ;

背景图片

background-image: none | url(url) ;常见于logo，一些装饰性的小图片或者是超大的背景图片。

背景平铺

background-repeat: repeat | no-repeat | repeat-x | repeat-y ;

背景位置

  1. 参数是方位名词

background-position: x y | top | center | bottom | left | center | right ;

两个方位名词，顺序无所谓。一个方位名词，另一个默认center。

  1. 参数是精确单位

两个参数第一个是x轴，第二个是y轴。只有一个参数，为x轴，另一个为center，垂直居中。

  1. 混合单位

第一个是x轴，第二个是y轴。

背景固定

background-attachment: scroll(随对象内容滚动) | fixed(固定) ;

背景复合写法

background: color image repeat attachment position ;

背景色半透明

background: rgba();red,green,blue,alpha(透明度)。0.3可以写作.3 。

CSS三大属性

层叠性

主要解决样式冲突的问题。层叠性原则：

  1. 样式冲突，遵循原则是就近原则。
  2. 样式不冲突，不会层叠。

继承性

子标签会继承父标签的某些样式(text-，font-，line-，color)，如文本颜色和字号等。

行高的继承
  • 行高可以跟单位也可以不跟单位

优先级

  • 选择器相同，执行层叠性

  • 选择器不同

    选择器 权重
    继承，* 0，0，0，0
    元素选择器 0，0，0，1
    类选择器，伪类选择器 0，0，1，0
    ID选择器 0，1，0，0
    行内样式style="" 1，0，0，0
    !important ∞

注意点：继承的权重为0。所以看标签样式首先看当前标签是否被选出来。

最后结果li中文字颜色是blue，因为 ul li 权重（0，0，0，2），li 权重（0，0，0，1），.nav li 权重（0，0，1，1）。.nav li > ul li > li。

盒子模型

边框

属性 作用
border-width 定义边框粗细
border-style 边框的样式：solid实线 dashed虚线 dotted点线
border-color 边框颜色

border: border-width border-style border-color 边框复合写法，无顺序。

border-collapse: collapse 合并相邻边框。

内边距

复合写法：

值
padding: 5px; 上下左右5
padding: 5px 10px; 上下5，左右10
padding: 5px 10px 15px; 上5，左右10，下15
padding: 5px 10px 15px 20px; 上5，右10，下15，左20

内边距也会影响盒子大小。

当盒子不设置宽度(width)，设置padding时不会撑大盒子。

外边距

margin复合写法与padding一致。

块级盒子水平居中：

  1. 盒子指定宽度width
  2. 左右外边距automargin: 0 auto;
垂直外边距的塌陷

解决方案：

  1. 为父元素定义上边框
  2. 为父元素定义上内边距
  3. 为父元素添加 overflow:hidden（不会额外增加宽度）

案例：通过内边距确定盒子大小

清除内外边距

网页元素很多带有默认的内外边距。

注意，行内元素为了照顾兼容性，尽量只设置左右内外边距，不要设置上下内外边距。但是转换为块级和行内块元素就可以了。

圆角边框

border-radius: length(%|px);：length越大，圆角越大。

border-radius: 50%;：正圆。

border-radius: height/2;：圆角矩形。

border-radius: 左上 右上 右下 左下;：分别设置四个角。

盒子阴影

box-shadow: h-shadow v-shadow blur spread color inset;

值 描述
h-shadow 必需。水平阴影位置。允许负值。
v-shadow 必需。垂直阴影位置。允许负值。
blur 可选。模糊距离。
spread 可选。阴影尺寸。
color 可选。阴影颜色。默认black.
inset 可选。将外部阴影(outset)改为内部阴影。

注意：

  1. 默认为outset，但是不可以写这个单词，会导致无效。

  2. 盒子阴影不占用空间，不会影响其他盒子排列。

文字阴影

浮动

CSS传统布局

  • 普通流（标准流）：标签按照规定好的默认方式排列。
  • 浮动
  • 定位

浮动属性

属性值 描述
none 不浮动
left 向左浮动
right 向右浮动

浮动框一直移动到左边缘或右边缘及包含块或另一个浮动框的边缘。

浮动特性

  1. 脱标：浮动元素会脱离标准流，脱标后，不再保留原来的位置。

  2. 按照属性值一行内显示并且顶端对齐排列。

  3. 浮动元素具有行内块元素特点：如果行内元素有了浮动，则不需要转换块级，行内块元素，可以直接给高度和宽度。

    如果块级没有设置宽度 ，默认宽度和父级一样宽，但添加浮动后，大小根据内容来决定。

注意点：

  1. 浮动和标准流的父盒子搭配

先用标准流父元素排列上下位置，之后内部子元素浮动排列左右位置，符合网页布局第一准则。

  1. 一个元素浮动了，理论上其余的兄弟元素也要浮动。

浮动的盒子只会影响浮动盒子后面的标准流，不会影响前面的标准流。

清除浮动

本质是清除浮动元素造成的影响。如果父盒子本身有高度，则不需要清除浮动。清除浮动后，父级就会根据子盒子自动检测高度。父级有了高度，就不会影响下面的标准流了。

clear: 属性值;

属性值 描述
left 不允许左侧有浮动元素
right 不允许右侧有浮动元素
both 同时清除左右两侧的浮动元素

策略：闭合浮动。

清除浮动方法
  1. 额外标签法（隔墙法）

W3C推荐方法。

添加<div style="clear: both;"></div>隔墙。注意到，这个新增的盒子要求必须是块级元素，不能是行内元素。

  1. 父级添加overflow属性

overflow 属性的属性值设置为 hidden 、 auto 或 scroll 。

overflow: hidden;：但是无法显示溢出部分。

  1. 父级添加after伪元素

css添加了类clearfix：

和隔墙法原理相同。

  1. 父级添加双伪元素

CSS属性书写顺序

  1. 布局定位属性：display，position，float，clear，visibility，overflow
  2. 自身属性：width，height，margin，padding，border，background
  3. 文本属性：color，font，text-decoration，text-align，white-space，break-down
  4. 其他属性（CSS3）：content，cursor，border-radius，box-shadow，text-shadow，background:linear-gradient...

定位

定位可以让盒子自由的在某个盒子内移动位置或者固定屏幕中某个位置，并且可以压住其他盒子。

组成

定位 = 定位模式 + 边偏移。

定位模式

通过position属性来设置。

值 语义
static 静态定位
relative 相对定位
absolute 绝对定位
fixed 固定定位

边偏移

边偏移属性 示例 描述
top top: 80px; 顶部偏移量，定义元素相对于其父元素上边线的距离
bottom bottom: 80px; 底部偏移量
left left: 80px; 左侧偏移量
right right: 80px; 右侧偏移量
静态定位 static

元素默认定位方式，无定位的意思。按照标准流特性摆放位置，没有边偏移。

相对定位 relative
  1. 相对自己原来的位置移动的（移动参照点是自己原来的位置）。
  2. 原来在标准流里的位置继续占有，后面的盒子依然以标准流的方式对待它（不脱标，继续保留原来位置）。
绝对定位 absolute
  1. 如果没有祖先元素或者祖先元素没有定位，则以浏览器为准定位。
  2. 如果祖先元素有定位（相对、绝对、固定定位），则以最近一级的有定位祖先元素为参考点移动位置。
  3. 不再占有原来位置。

子绝父相

子元素是绝对定位，父元素就得是相对定位。

固定定位 fixed

元素固定于浏览器可视区的位置。

  1. 以浏览器的可视窗口为参照点移动元素：跟父元素没有任何关系，不随滚动条滚动。
  2. 不占有原先的位置。

固定定位小技巧：固定版心右侧位置

  1. left: 50%; 放置可视区一半位置。
  2. margin-left: 版心距离一半位置；多走版心宽度一半位置。
粘性定位 sticky
  1. 以浏览器的可视窗口为参照点移动元素（固定定位特点）
  2. 占有原先位置（相对定位特点）
  3. 必须添加top/left/right/bottom其中一个才有效

叠放次序

使用z-index控制盒子的前后顺序。

  1. 属性值可以是任意整数，默认auto，数值越大，盒子越靠上。

  2. 如果属性值相同，按照书写顺序，后来者居上。

  3. 只有定位的盒子才有z-index属性

定位特殊特性

  • 绝对定位和固定定位与浮动类似。
  1. 行内元素添加绝对或者固定定位，可以直接设置高度和宽度。
  2. 块级元素加绝对或者固定定位，如果不给宽度或高度，默认大小是内容的大小。

  • 浮动元素、绝对定位（固定定位）元素的都不会触发外边距合并的问题。

  • 绝对定位会完全压住盒子。

浮动元素只会压住它下面标准流的盒子，不会压住下面标准流盒子里面的文字（图片）。

但是绝对定位（固定定位）会压住下面标准流所有的内容。

元素的显示与隐藏

display

属性值 含义
none 隐藏对象
block 除了转换为块元素，还有显示元素的含义
...

隐藏后，不再占有原来位置。

visibility

属性值 含义
inherit 继承上一父对象的可见性
visible 对象可视
collapse 对象隐藏
hidden 用来隐藏列表的行或列

隐藏后，继续占有原来位置。

overflow

属性值 含义
visible 默认值，不剪切内容，不添加滚动条
auto 剪切，添加滚动条
hidden 不显示超过对象尺寸的内容
scroll 总是显示滚动条
